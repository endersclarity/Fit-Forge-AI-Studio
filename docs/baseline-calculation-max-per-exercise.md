# Muscle Baselines - Maximum Single-Exercise Volume Approach

## Methodology
For each muscle group, find the exercise that produces the HIGHEST volume for that muscle when performed in isolation.

## FINAL RECOMMENDED BASELINES

Based on highest single-exercise volume for each muscle:

```
Pectoralis:   2,700 lb  (Dumbbell Bench Press)
Triceps:      1,300 lb  (TRX Dips)
Deltoids:     1,200 lb  (TRX Pushups / Upright Row)
Lats:         2,100 lb  (TRX Pull-ups)
Biceps:       1,600 lb  (TRX Pull-ups)
Rhomboids:    1,200 lb  (Dumbbell Row)
Trapezius:    2,400 lb  (Shoulder Shrugs)
Forearms:       600 lb  (TRX Pull-ups)
Quadriceps:   4,800 lb  (Box Step-ups)
Glutes:       4,400 lb  (Stiff-Leg Deadlift)
Hamstrings:   4,700 lb  (Romanian Deadlift)
Calves:       5,500 lb  (Calf Raises)
Core:         3,200 lb  (Stiff-Leg Deadlift)
```

These represent conservative baselines - the maximum volume each muscle experiences from a single exercise. Real workout sessions combining 2-3 exercises per muscle would likely exceed these values, allowing the app to learn and increase baselines over time.
